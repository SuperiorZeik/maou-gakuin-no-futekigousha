# 3-6 蓋拉帝提遠征測驗



隔周──德魯佐蓋多第二訓練場。

<br/>
<br/>
在此集合的學生們各個都帶著大包小包的行李。武器與魔法具就不用說，還在大皮袋裏塞滿了糧食、替換衣物和飲用水等物品，一副就像是要出外旅遊的樣子。

<br/>
<br/>
當上課鍾聲響起時，窗外就飛來了一只貓頭鷹。

<br/>
<br/>
「各位同學大家早。」

<br/>
<br/>
貓頭鷹發出梅諾的聲音。她以貓頭鷹爲媒介，用魔法在和我們說話。

<br/>
<br/>
「那麽，今天終于要前往亞傑希翁進行學院交流了唷。目的地是王都蓋拉帝提。就跟事前通知的一樣，這次遠行學院不會派教師帶隊。雖然三年級的同學們都已經知道了，不過爲了一年級生，老師就再說明一次。」

<br/>
<br/>
大概有別只貓頭鷹在三年級的教室裏，也同樣用梅諾的聲音說話吧。

<br/>
<br/>
「德魯佐蓋多在進行這種遠行時，幾乎都不會有教師帶隊。因爲想要成爲魔皇的人，怎麽能不靠自己的力量抵達目的地呢？啊，不過學生之間互助合作當然是沒問題的唷。」

<br/>
<br/>
唔，就連抵達目的地都是課程的一部分啊？

<br/>
<br/>
「在從學院前往亞傑希翁的路上有各式各樣的障礙喔。如果要走海路，就得越過艾路凱海峽，走陸路的話，就必須翻越迪魯鐵斯特山脈，或是繞道穿越托拉之森。而就算要走空路飛過去，也由于亞傑希翁空域的魔力場紊亂，所以不是一件簡單的事喔。」

<br/>
<br/>
要走哪一條路線會比較快，這當然也考驗到選擇適合自己路線的智慧。

<br/>
<br/>
「亞傑希翁不同于迪魯海德，各位同學肯定能在旅途上遇到許多前所未見的事物。不只是學院所教授的知識，也希望各位同學能在這次遠行過程中學習到對應未知事物的方法。」

<br/>
<br/>
與以往的課程相比，似乎還挺有趣的呢。只可惜不論是艾路凱海峽、迪魯鐵斯特山脈，還是托拉之森，我都已去過好幾遍了。

<br/>
<br/>
不過，景色大概也跟兩千年前有所不同了吧。

<br/>
<br/>
「期限是十天。沒能在期限內抵達位在蓋拉帝提的勇者學院第三宿舍的學生，就沒有資格參與這次的學院交流。當然，抵達的名次以及抵達所費的時間也都會影響到成績，因此各位要好好努力喔。三年級生已經習慣這種遠行了，所以一年級生要是比他們更快抵達，得分會很高唷。」

<br/>
<br/>
十天啊？也好，算是妥當的期限呢。即便是走陸路過去，也只要以一定的速度持續奔跑就能從容抵達。

<br/>
<br/>
「順道一提，老師兩天就到了喔。三年級生也要以此爲基准。」

<br/>
<br/>
兩天啊？不愧是教師，速度還算可以。考慮到她是三年級生的教師，說不定比艾米莉亞還要優秀。

<br/>
<br/>
「那麽，蓋拉帝提遠征測驗即將開始了唷。預備──開始！」

<br/>
<br/>
在她的口令之下，第二訓練場的學生約有半數一齊往外沖。

<br/>
<br/>
其余的人則是一面確認地圖，一面討論要走哪條路線。

<br/>
<br/>
「喂，阿諾斯，我從剛才就很在意了，你真的什麽東西都沒帶來耶。是放進收納魔法陣裏了嗎？」

<br/>
<br/>
莎夏向我問道。她的行李也很少。大概是除了魔法具這類放在手邊會比較好的東西外，其余全都收進收納魔法陣裏了吧。

<br/>
<br/>
「沒什麽，我不需要行李。以前經常當天來回蓋拉帝提。」

<br/>
<br/>
畢竟只要發現人類在搞小動作，就得過去一一處理掉才行。

<br/>
<br/>
「當天來回……你還是一樣誇張耶……」

<br/>
<br/>
「要走哪個路線？」

<br/>
<br/>
米夏問道。

<br/>
<br/>
「就空路吧。就算說亞傑希翁空域的魔力場紊亂，也總會有辦法的吧。或是說，魔力場紊亂這種事，已經在不知道是誰的自習課中充分體驗過了……」

<br/>
<br/>
莎夏偏好使用「飛行」，所以我就在自習時讓她稍微難飛了一點。大概是因爲這種訓練的成果吧，使得她連在魔力狂亂的環境下，也能飛得相當好。

<br/>
<br/>
「雷伊和米莎呢？」

<br/>
<br/>
米夏看向兩人。

<br/>
<br/>
「啊哈哈，我雖然不會飛，但只要施展『雨靈霧消』，我想就能趕上期限了。」

<br/>
<br/>
米沙能在「雨靈霧消」的範圍內自由移動。盡管這麽做的效率有點差，但只要連續施展，或許就能維持一定的移動速度了吧。

<br/>
<br/>
「要我選的話，還是用跑的會比較快。我不太擅長『飛行』。」

<br/>
<br/>
「那就走陸路吧？只要我低空飛行就好。這樣大家就能一起走了。」

<br/>
<br/>
莎夏用魔法當場畫起地圖，再用紅線標出三條遠行路線。

<br/>
<br/>
「如果要穿越最短距離的托拉之森，我認爲可以走這三條路線。當中最快的路線，是這條穿越米列奴沙漠的路線。當然，這是指路上沒發生意外的情況。但要是順利的話，只要一天就能抵達了喔。」

<br/>
<br/>
因爲梅諾說三年級生要以兩天爲基准，所以激起她的競爭心了吧。

<br/>
<br/>
很有莎夏的風格。

<br/>
<br/>
「莎夏，如果打算用走的話，這條路線確實不錯呢。但你們要去蓋拉帝提，把地圖攤開來是要做什麽啊？」

<br/>
<br/>
聽我這麽一問，莎夏就露出錯愕的表情。

<br/>
<br/>
「我方才說過我曾去過了吧？你以爲我在回家時用的魔法是什麽啊？」

<br/>
<br/>
莎夏「啊」了一聲。

<br/>
<br/>
「『轉移』……？」

<br/>
<br/>
米夏問道。

<br/>
<br/>
「這麽長的距離也能施展嗎？」

<br/>
<br/>
「用現代的說法來說，『轉移』可是爲了每天必須進行好幾趟長距離移動的繁忙魔族所開發出來的魔法。」

<br/>
<br/>
「……這點我懂，但你不需要改用現代的說法。」

<br/>
<br/>
我好心用淺顯易懂的方式說明耶。

<br/>
<br/>
「總之，事情就是這樣。別說一天，一秒就到了。」

<br/>
<br/>
我把手伸向莎夏，她一臉半信半疑地牽起我的手。

<br/>
<br/>
當全員把手牽起後，我就施展了「轉移」。轉瞬間，染成純白一片的風景取回色彩，眼前出現一片廣大湖泊。後方豎立著城牆，城門後是一望無際的街景。

<br/>
<br/>
王都蓋拉帝提是以湖泊爲中心建造的要塞都市。這座傳說中會湧出聖水的聖明湖本身形成自然魔法陣的作用，具備封魔之力。經過兩千年後，如今也仍有聖水湧出的樣子，但就不知道這個時代還有沒有人能運用自如了？

<br/>
<br/>
「像這樣一下子就到了，還真是煞風景呢。」

<br/>
<br/>
「聽說勇者都擅長使劍，要是能在學院交流前遇到，似乎會很有趣呢。」

<br/>
<br/>
聽雷伊這麽說，莎夏就一臉傻眼地看著他。就算能在學院交流前遇到勇者，你是打算怎樣讓對方答應跟你比劍啊──她心裏是怎麽想的吧。

<br/>
<br/>
「雷伊成天都在想劍的事情耶。」

<br/>
<br/>
聞言，他就爽朗地笑著說：

<br/>
<br/>
「或許吧。就跟你成天都在想魔王的事情差不多呢。」

<br/>
<br/>
「你……你在說什麽啊……」

<br/>
<br/>
就像動搖似的，莎夏整張臉頓時紅成一片。

<br/>
<br/>
「魔王學院的學生就算成天都在想魔王的事情，也沒什麽問題吧？畢竟你好像是位優等生呢。」

<br/>
<br/>
一臉氣憤的莎夏狠狠瞪著雷伊。

<br/>
<br/>
「…………給我記住，你這劍癡……」

<br/>
<br/>
莎夏與雷伊很難得地鬥起嘴來。

<br/>
<br/>
我們就這樣走向城門。站在入口處的士兵在確認過我們身上的制服與校徽後，隨即放我們通行。在穿越城門的途中，身後傳來「不是說今天還不會到嗎？」的詫異聲。

<br/>
<br/>
「話說回來，勇者學院的第三宿舍在哪裏啊？」

<br/>
<br/>
「在勇者學院的正東方，壁壘旁邊。」

<br/>
<br/>
米夏簡潔說道。

<br/>
<br/>
「唔，勇者學院是哪一棟啊？」

<br/>
<br/>
「阿諾斯也不知道？」

<br/>
<br/>
「因爲兩千年前沒有嘛。」

<br/>
<br/>
隨後，米夏就指向遠處一棟高聳的建築物。

<br/>
<br/>
「是那個，勇者學院亞魯特萊茵斯卡。拿到的資料上有寫。」

<br/>
<br/>
唔，是將兩千年前的王城亞魯特萊茵斯卡改建爲勇者學院啊？在世界和平之後，盡管不再需要軍事設施，卻覺得失去這些魔法設備很可惜，所以就作爲教育場所有效運用吧。畢竟德魯佐蓋多也改建爲魔王學院了，這件事沒有不自然之處。

<br/>
<br/>
只不過，乍看之下沒有不自然之處，反倒讓人感到可疑。

<br/>
<br/>
「總之先過去吧。都特意第一個到了，得去找梅諾老師報到才行。」

<br/>
<br/>
「可是，老師會在宿舍嗎？到底是認爲今天還不會有人到吧？」

<br/>
<br/>
「……這麽說也是呢……」

<br/>
<br/>
我們邊這樣討論，邊朝勇者學院第三宿舍走去。

<br/>
<br/>
順道遊覽陌生的街景走了一會後，眼前就看到一棟富麗堂皇的石造建築物。此建築占地相當遼闊，房間數大概能住進兩百多人，門上寫著亞魯特萊茵斯卡第三宿舍。

<br/>
<br/>
「找到了。」

<br/>
<br/>
米夏一邊說一邊伸手指著。這時梅諾剛好從宿舍走出來。

<br/>
<br/>
「唔，梅諾，我們到了喔。」

<br/>
<br/>
「…………………………咦……？」

<br/>
<br/>
梅諾看著我們，彷佛時間暫停似的渾身僵住。

<br/>
<br/>
「紀錄還算可以吧？」

<br/>
<br/>
我朝她說道，但梅諾就像是愣住似的聽而不聞。大概是在想什麽事情吧，她眨著眼睛盯著我們數秒後，才總算開口。

<br/>
<br/>
「……等、等等……騙人的吧……？就連一天……別說一天了，就連一個小時都還沒過耶……你們，到底是怎麽過來這裏的……！」

<br/>
<br/>
她一副難以置信的模樣滔滔不絕地說。大概是因爲她已經透過貓頭鷹的魔眼視覺確認過我們人在迪魯海德了吧。這件事沒有懷疑我們作弊的余地。

<br/>
<br/>
「我施展了『轉移』。因爲以前曾來過蓋拉帝提。」

<br/>
<br/>
「……雖然聽說過你會施展失傳的魔法……可是，居然能將相隔如此遙遠的空間連接起來，真是難以置信……」

<br/>
<br/>
梅諾就跟她說的一樣，露出目瞪口呆的表情。

<br/>
<br/>
「……我知道你有魔法的才能……我也是教師，至今也曾看過許多人稱天才的孩子。即使是混血，也有許多魔法天資優秀的學生。可是，你這已經不是能用天才這種尋常話語形容的層級了。」

<br/>
<br/>
梅諾用她的魔眼直盯著我的臉，開口問道：

<br/>
<br/>
「阿諾斯同學，你……究竟是什麽人？」

<br/>
<br/>
「我已經說過很多次了。要是不相信自己的魔眼，一味相信他人的話語的話，可是一輩子都抵達不了真理的喔。」

<br/>
<br/>
暴虐魔王──或許是腦海中閃過了這個名字吧，梅諾沈默不語。
