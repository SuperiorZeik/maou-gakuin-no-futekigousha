# 3-30 禁忌的魔法


「那麽，今天的課程就到此爲止。」

<br/>
<br/>
結果直到放學後，艾蓮歐諾露都沒有出現。認爲她發生什麽事，幾乎不會錯了吧。學院對抗測驗時，待在神殿裏的艾蓮歐諾露曾說她無法靠自己的意思行動。如果她處于跟當時相同的狀態，就沒辦法出現在這裏了吧。

<br/>
<br/>
那我就自己去見她吧。

<br/>
<br/>
「那個，阿諾斯大人。接下來大家打算一起去逛祭典，不知您意下如何……？」

<br/>
<br/>
米莎向我問道。

<br/>
<br/>
「唔，我稍微有點事。你們好好去玩吧。」

<br/>
<br/>
「這樣啊。」

<br/>
<br/>
「我今天也不去了。」

<br/>
<br/>
雷伊說道。

<br/>
<br/>
「我有點想睡。」

<br/>
<br/>
「是熬夜了嗎？」

<br/>
<br/>
「換了枕頭，有點睡不著呢。」

<br/>
<br/>
「這樣啊……」

<br/>
<br/>
米莎很遺憾的樣子。雷伊靠過去，和她咬耳朵說：

<br/>
<br/>
「等我小睡一下後，我們再會合吧。」

<br/>
<br/>
「……咦？什麽……」

<br/>
<br/>
原來如此。想兩個人偷偷地單獨相處啊？還真是惹人微笑。

<br/>
<br/>
「討厭嗎？」

<br/>
<br/>
「不、不是的。那麽，好的。就、就這樣決定了。」

<br/>
<br/>
看著這樣的兩人，粉絲社的少女們竊竊私語。

<br/>
<br/>
「……雷伊同學說他想睡耶……」

<br/>
<br/>
「嗯。雷伊同學和阿諾斯大人在宿舍是睡同一間房吧……？」

<br/>
<br/>
「等等、等等！你在想什麽啊！」

<br/>
<br/>
「是沒在想什麽，就只是關于特殊接觸稍微……」

<br/>
<br/>
「別說什麽特殊接觸啦！」

<br/>
<br/>
「……那、那麽，也就是說……今天……米莎她……」

<br/>
<br/>
「是間接同床的機會！」

<br/>
<br/>
我無視她們的對話，離開大講堂。

<br/>
<br/>
我啓動魔眼，試著找尋艾蓮歐諾露的魔力卻找不到。是消去魔力了嗎？這麽說來，昨天米夏也沒辦法找出艾蓮歐諾露的位置。

<br/>
<br/>
不過，倒是能捕捉到迪耶哥的魔力。要是勇者學院對她做了什麽，我不認爲那位學院長會一無所知。

<br/>
<br/>
我施展「幻影擬態」透明化，再用「隱匿魔力」隱藏魔力後，尾隨著方才離開學院的迪耶哥。

<br/>
<br/>
他快步前往的地方，是聖明湖。湖水依舊是被蒸發殆盡的模樣。迪耶哥施展「飛行」降落到湖底。原以爲他要去神殿，結果卻是前往反方向的水中洞窟。

<br/>
<br/>
走進昏暗的洞窟裏，深處有一池小泉水。那是湧出的聖水。他施展「水中活動」的魔法，跳進那池泉水裏。

<br/>
<br/>
試著跟上後，發現泉水裏的情況跟外面看起來不同，有著相當深廣的空間。迪耶哥在裏頭施展「飛行」，有如遊泳般的飛翔，並不斷朝水底潛去。

<br/>
<br/>
不知道潛了多深後，才總算看到水底。那裏有一扇巨大門扉。看樣子是施加了「施鎖結界」的魔法。允許通行的迪耶哥開門入內。

<br/>
<br/>
要是跟著開門進去，就算消去了身影與魔力，也不免還是會被察覺。

<br/>
<br/>
我稍待片刻等他遠離之後，解除「施鎖結界」開門入內。

<br/>
<br/>
門後是石造的建築。似乎施展了魔法，讓泉水不會滲到門後的樣子。

<br/>
<br/>
他究竟在這裏做什麽？我邊思考邊走在通道上。隨後，便發現到一處牆壁崩坍的地方。

<br/>
<br/>
痕迹比較新。從未經修理的情況來看，是在這幾天崩坍的吧。

<br/>
<br/>
愈是往內部前進，遭到破壞的地方就愈是明顯。地板、天花板，還有牆壁都遭人打碎或斬斷，開出好幾個破洞。彷佛最近這裏發生過戰鬥一樣。

<br/>
<br/>
「還沒找到任何線索嗎！」

<br/>
<br/>
突然傳來一道怒吼。那是迪耶哥的聲音。從附近的門後傳來的。

<br/>
<br/>
我站在門前豎起耳朵。

<br/>
<br/>
「……只、只知道是帶著面具的男人……」

<br/>
<br/>
「這我早上聽過報告了！我要你給我新的情報！」

<br/>
<br/>
「真是非常抱歉。」

<br/>
<br/>
「是魔王學院的人幹的嗎？」

<br/>
<br/>
「……關于這點，由于偵測不到賊人的魔力，就連是否爲魔族所爲都無法辨別……」

<br/>
<br/>
偵測不到魔力的面具男啊……好像在哪裏聽過呢。

<br/>
<br/>
「而且我們的計畫沒有泄露給魔王學院知道。他們會攻擊這座設施的可能性應該很低。屬下認爲這恐怕是西方的殷茲艾爾帝國所爲。」

<br/>
<br/>
「他們應該沒有這麽做的理由！我們可是建立了千年以上的友好關系喔！」

<br/>
<br/>
「……話雖如此，但對方也沒笨到會完全信任蓋拉帝提吧。會不會是內部有密探潛入？或許他們偶然探聽到聖母的傳聞。」

<br/>
<br/>
大概是迪耶哥不發一語吧，門後沈寂下來。

<br/>
<br/>
「沒讓賊人察覺到聖母的所在位置吧？」

<br/>
<br/>
「是的。雖然面具男大鬧了一場，但看來是沒讓他找到的樣子。」

<br/>
<br/>
隨後，迪耶哥就一副想到好主意的感覺說道：

<br/>
<br/>
「很好，那就這麽辦吧。把這次的襲擊設計成是魔族幹的。」

<br/>
<br/>
「……要捉一名魔王學院的學生來嗎？」

<br/>
<br/>
「就算不擇手段也無所謂。只要能讓民衆知道正義是在我們這裏就好。制造一個能讓我們進攻迪魯海德的正當名義。」

<br/>
<br/>
「那麽，終于？」

<br/>
<br/>
「實現我等夙願的時刻到了。」

<br/>
<br/>
「遵命！屬下知道了。」

<br/>
<br/>
「密探的事情就交給你處理。給我找出來，讓他把事情全盤托出。就算不擇手段也要達成目的。」

<br/>
<br/>
「遵命！」

<br/>
<br/>
真是愚蠢。不惜捏造莫須有的事端也想開戰啊？

<br/>
<br/>
明明難得迎來了和平，到底爲什麽會這麽不滿？盡管可以在這裏殺掉他，不過昨天也才剛消滅掉他的根源，還是阻止他們的計畫比較快吧。

<br/>
<br/>
盡管不知道聖母是指什麽，但應該是勇者學院的重要人物吧。

<br/>
<br/>
就從至今爲止的發展來看，說不定就是艾蓮歐諾露。

<br/>
<br/>
既然面具男找不到，就表示這裏的某處應該有隱藏房間。但就算我啓動魔眼，也看不出施有魔法機關的地方。說到底，如果是魔法機關的話，應該會被他找出來吧。這樣一來，應該就跟魔王城的地城一樣，是設有不帶魔力的隱藏房間。

<br/>
<br/>
我一度折返，回到牆壁被破壞的通道上。

<br/>
<br/>
微微擡腳，咚地一聲用力踏地。緊接著一陣轟隆隆隆隆隆隆的巨響，我踏地的震動讓整棟建築物劇烈搖晃。

<br/>
<br/>
「敵、敵襲！全隊就戰鬥位置！」

<br/>
<br/>
伴隨著緊迫的吶喊聲，士兵們亂糟糟地蜂湧而出。不過在注意到哪裏都沒有賊人身影之後，全都露出詫異的表情。

<br/>
<br/>
「……會是地震嗎？……」

<br/>
<br/>
「……聖明湖應該不常發生地震……是湖水乾枯的影響嗎？……」

<br/>
<br/>
士兵們進行著這種對話。我趁機用魔眼調查他們的魔力，掌握住全隊的部署位置。

<br/>
<br/>
唔，是在那裏啊？我經由通道朝目標地點走去。

<br/>
<br/>
稍待片刻，等士兵們都離開之後，我面向一道平凡無奇的牆壁，將指尖抵在牆上壓下去。牆壁隨即緩緩開啓。是一道不用魔力的密門。

<br/>
<br/>
就算再怎麽避免情報外流，人性都會在緊急時自然而然地去保護必須保護的東西。只要將士兵的部署位置與搖晃建築物時的不自然震動互相對照，就會知道答案了。

<br/>
<br/>
我沿著門後的昏暗道路直走下去。沿途設置了許多同樣沒用魔力的陷阱，但全都被我輕易識破。不久後，前方冒出微微藍光。

<br/>
<br/>
那裏是個寬廣的空間，飄浮著上千，不對，是上萬顆的聖水球，而裏頭全都裝著一名全裸的少女。

<br/>
<br/>
是潔西雅．加隆．伊捷伊西卡。毫無疑問，她正是在學院對抗測驗中跟我交戰過的那位排行第一位。人數上萬的少女，全都擁有完全相同的根源。

<br/>
<br/>
而在房間中央的巨大聖水球裏，則裝著艾蓮歐諾露。

<br/>
<br/>
就跟昨天在神殿裏看到的時候一樣，她的全身就像在釋放魔力地發光，讓整個人的輪廓很模糊。許多魔法文字就像纏繞在這副身軀上似的，飄浮在她的周圍。能看出她的魔力正流向其他的聖水球。

<br/>
<br/>
「艾蓮歐諾露。」

<br/>
<br/>
我解除「幻影擬態」，呼喊她的名字。

<br/>
<br/>
「……阿諾斯弟弟……」

<br/>
<br/>
她又驚又喜地注視著我。

<br/>
<br/>
「抱歉，我沒辦法去學校。這種情況也有點出乎我的意料之外。」

<br/>
<br/>
「沒什麽，就跟約好的，一樣是在放學後見面。一點問題也沒有。」

<br/>
<br/>
語罷，艾蓮歐諾露綻開微笑。

<br/>
<br/>
「我就覺得你一定會來喔。」

<br/>
<br/>
她豎起食指說道。然後，露出溫柔的表情注視著我。

<br/>
<br/>
「雖然是在這種地方，但你能聽一下我的心願嗎？」

<br/>
<br/>
「說吧。」

<br/>
<br/>
「小弟想請阿諾斯弟弟消滅一個人的根源。」

<br/>
<br/>
「唔，誰的？」

<br/>
<br/>
她以毫不顧慮的語調說道：

<br/>
<br/>
「我的。」

<br/>
<br/>
艾蓮歐諾露露出毫不虛假的笑容，彷佛在述說，這是她由衷的心願。

<br/>
<br/>
「我一直在等待，能從這個永無止盡的地獄之中，讓我和潔西雅獲得解放的人。阿諾斯弟弟，我呢──」

<br/>
<br/>
然後，她坦白說：

<br/>
<br/>
「是人不該創造出來──禁忌的魔法唷。」
