# 3-1 某天早晨的平穩生活



黑暗之中，感覺有只小手碰觸我。

<br/>
<br/>
「阿諾斯。」

<br/>
<br/>
聽到少女耳熟的呼喚，身體被輕輕搖晃。

<br/>
<br/>
「早餐。」

<br/>
<br/>
我靜靜地睜開眼，一張少女的臉蛋出現在眼前。留長的白金色側發輕輕搔過鼻頭，清澈的綠眼眨了兩下。是米夏。

<br/>
<br/>
「醒了？」

<br/>
<br/>
「是啊。」

<br/>
<br/>
我簡潔回應後，米夏愉快地笑起。

<br/>
<br/>
「早安。」

<br/>
<br/>
我緩緩離開床鋪，一面在腳下畫起魔法陣，一面向她問道：

<br/>
<br/>
「米夏怎麽會在這裏？」

<br/>
<br/>
魔法陣從腳邊上升到頭頂，將身上的睡衣換成制服。

<br/>
<br/>
「今天要練習做便當。」

<br/>
<br/>
原來如此。是趁媽媽幫我做便當時，順便請她指導啊？

<br/>
<br/>
「也做了早餐。」

<br/>
<br/>
「還真是期待呢。」

<br/>
<br/>
我這麽說完後，米夏便有點驚訝地眨了眨眼。

<br/>
<br/>
「怎麽了嗎？」

<br/>
<br/>
「要吃嗎？」

<br/>
<br/>
「是說早餐嗎？」

<br/>
<br/>
米夏點了點頭，然後指著自己。

<br/>
<br/>
「我做的？」

<br/>
<br/>
「沒有我的份嗎？」

<br/>
<br/>
「媽媽做了阿諾斯的早餐。」

<br/>
<br/>
是這個意思啊？

<br/>
<br/>
「那我吃媽媽做的也無妨。」

<br/>
<br/>
「嗯。」

<br/>
<br/>
米夏就跟往常一樣淡然回應，推開房門。盡管依舊面無表情，但總覺得她好像很沮喪。

<br/>
<br/>
「哎，不過，要是能跟米夏做的早餐交換，那我會很高興喔。」

<br/>
<br/>
她就像在揣摩我的用意似的直盯著我的眼睛。

<br/>
<br/>
「……可以嗎？」

<br/>
<br/>
「你願意的話。」

<br/>
<br/>
米夏想了一下後說道：

<br/>
<br/>
「阿諾斯喜歡媽媽的料理。」

<br/>
<br/>
「是這樣沒錯，但米夏的料理只有偶爾才能吃到。」

<br/>
<br/>
她稍微垂下頭，看起來很高興，卻又覺得不好意思。

<br/>
<br/>
「真溫柔。」

<br/>
<br/>
「一時興起罷了。」

<br/>
<br/>
米夏忙不叠地搖頭。

<br/>
<br/>
「阿諾斯懂嗎？」

<br/>
<br/>
「懂什麽？」

<br/>
<br/>
「我的心情。」

<br/>
<br/>
「看起來有點失望吧？」

<br/>
<br/>
經我指出，米夏就微微垂下眼簾。

<br/>
<br/>
「……好害羞……」

<br/>
<br/>
「你把我看得很透徹。」

<br/>
<br/>
在看穿我內心這方面，說不定無人能出其右。

<br/>
<br/>
「不過，我的魔眼可不會輸給你喔。」

<br/>
<br/>
聽我這麽一說，米夏稍微瞠圓了眼，呵呵地笑了起來。

<br/>
<br/>
「我說了什麽奇怪的事嗎？」

<br/>
<br/>
「猜猜看。」

<br/>
<br/>
是要我猜她爲什麽會笑吧？

<br/>
<br/>
「很開心嗎？」

<br/>
<br/>
米夏露出微笑。

<br/>
<br/>
「再看清楚一點。」

<br/>
<br/>
米夏沒說正不正確，就這樣轉身走下一樓。我也跟在後頭，一起走向客廳。

<br/>
<br/>
餐桌上已備好早餐，但只有兩人份。

<br/>
<br/>
「爸爸和媽媽呢？」

<br/>
<br/>
「工作。」

<br/>
<br/>
這麽說來，爸爸在幫忙打造金剛鐵劍的工作室那邊還有事要幫忙。受到魔劍大會的影響，似乎讓對方對他另眼相看。盡管曾邀請他今後也務必要過去幫忙，但他目前似乎沒這個打算。

<br/>
<br/>
「媽媽呢？」

<br/>
<br/>
「有客人請她到府鑒定，是有點遠的地方。」

<br/>
<br/>
所以就提早出門啊？

<br/>
<br/>
「出門時說阿諾斯因爲魔劍大會累了，要我別叫醒你。」

<br/>
<br/>
雖然沒有很累，但很像是爸媽會說的話。

<br/>
<br/>
「那就開動吧。」

<br/>
<br/>
「嗯。」

<br/>
<br/>
我跟米夏就座，用起早餐。由于平時吵吵鬧鬧的爸媽不在，所以是個難得甯靜的早晨。用完餐後，我用魔法倏地收拾好餐具，離開家門。我們兩人並肩齊行，悠然走在通往魔王學院的道路上。

<br/>
<br/>
當然，只要施展「轉移」就能立刻抵達，但時間還很充裕。反正就算趕著上學也不會有獎品。一面眺望早晨的街景，一面慢步走在通學道路上，感覺也挺不錯的。

<br/>
<br/>
「咦……？」

<br/>
<br/>
在通學途中，突然遇到一個熟面孔。將一頭金發綁成雙馬尾，穿著魔王學院的黑制服。是莎夏。她用奇怪的眼神盯著我們兩人。

<br/>
<br/>
「……爲什麽你會和米夏兩個人一起上學？」

<br/>
<br/>
「就早上遇到。」

<br/>
<br/>
「我說啊，我當然知道你們是遇到的。不過請不要懶得說明就隨便回答好嗎？」

<br/>
<br/>
「便當。」

<br/>
<br/>
米夏說道。

<br/>
<br/>
「我請阿諾斯的媽媽教我做。」

<br/>
<br/>
「這樣啊？哼──這麽說來，你說過你在學習做料理呢。既然早上要去，怎麽不跟我說一聲呢？」

<br/>
<br/>
大概是覺得自己被排擠了吧，莎夏看起來有點不滿。

<br/>
<br/>
「說過了。」

<br/>
<br/>
「咦？什麽時候？」

<br/>
<br/>
「早上，出門前。」

<br/>
<br/>
莎夏就像在回想似的低著頭，一副完全沒有記憶的樣子。

<br/>
<br/>
「我起床時，米夏就已經不見人影了耶……？」

<br/>
<br/>
米夏忙不叠地搖頭。

<br/>
<br/>
「那是第二次。」

<br/>
<br/>
「騙人……？真的嗎……？」

<br/>
<br/>
也就是說，她跑去睡回籠覺了啊。

<br/>
<br/>
「唔，原來莎夏你早上爬不起來啊？」

<br/>
<br/>
「才沒這回事……」

<br/>
<br/>
我看向米夏，她頻頻點頭。

<br/>
<br/>
「很難起床。」

<br/>
<br/>
「我、我就只是有點離不開床鋪，醒來時腦袋一片空白，記憶會變得很模糊而已啦。」

<br/>
<br/>
這怎麽聽都覺得她很難起床吧？

<br/>
<br/>
「你是怎樣啦，那個得意的眼神。」

<br/>
<br/>
「沒關系，你不需要感到可恥。就算早上爬不起來，也不會造成任何障礙不是嗎？人生也不會因此完蛋。」

<br/>
<br/>
「別說得事情好像很嚴重一樣好嗎？」

<br/>
<br/>
我只是在說這沒什麽好可恥的，不懂她在生什麽氣。

<br/>
<br/>
「夠了，趕快上學吧。」

<br/>
<br/>
莎夏一邁開步伐，米夏就立刻快步追上。

<br/>
<br/>
「生氣了？」

<br/>
<br/>
「我生什麽氣啊？」

<br/>
<br/>
「……因爲我一個人去。」

<br/>
<br/>
「我才不在意這種事呢。就算大清早繞遠路到阿諾斯家裏，也沒什麽意義。」

<br/>
<br/>
米夏垂著頭，想了一會。

<br/>
<br/>
「……我不會再去了……」

<br/>
<br/>
「爲什麽會變成這樣啦！就說我不在意了吧？米夏想去的話就去啊。」

<br/>
<br/>
米夏像是很爲難似的低頭不語。我忍不住「咯哈哈」笑了起來。

<br/>
<br/>
「你、你笑什麽啊？」

<br/>
<br/>
「莎夏，才剛碰面你就謊話連篇呢。想來我家就坦白說啊。」

<br/>
<br/>
「我……我才沒說我想去呢……」

<br/>
<br/>
她愈說愈小聲。

<br/>
<br/>
「因爲早上爬不起來，想說反正沒辦法來，所以才在那邊賭氣說你不在意吧？沒關系，你放心好了。在我面前，早上爬不起來根本算不了什麽。」

<br/>
<br/>
「……那個……雖然你好像說得很誇張，但你打算怎麽做？」

<br/>
<br/>
「我直接去叫你起床。」

<br/>
<br/>
「……咦？」

<br/>
<br/>
莎夏整張臉頓時紅成一片。

<br/>
<br/>
「莎夏，你似乎很擅長睡回籠覺呢。但我可沒有米夏這麽好說話喔。在我面前，你是別想睡了。」

<br/>
<br/>
我邊這麽說，邊探頭注視著莎夏的臉。

<br/>
<br/>
「……啊……………」

<br/>
<br/>
「答覆呢？」

<br/>
<br/>
無法正視我的眼睛，莎夏的視線往斜下方望去。

<br/>
<br/>
「………………好的……」

<br/>
<br/>
莎夏以細如蚊鳴的聲音答覆。

<br/>
<br/>
唔，早上爬不起來有這麽可恥嗎？

<br/>
<br/>
「這樣下次就能一起來我家了喔。」

<br/>
<br/>
我朝米夏這麽說後，她就開心地點了點頭。

<br/>
<br/>
「……可、可是……總覺得不太對勁耶。明明是要去阿諾斯家，阿諾斯卻要特地來叫我起床……」

<br/>
<br/>
莎夏喃喃自語著。

<br/>
<br/>
「嗨，早啊。」

<br/>
<br/>
聽到爽朗的招呼，回頭就看到雷伊站在那裏。

<br/>
<br/>
「早，還真巧呢。」

<br/>
<br/>
米夏與莎夏也一齊道「早安」跟他打招呼。

<br/>
<br/>
「你們總是一塊上學嗎？」

<br/>
<br/>
「沒有，今天只是碰巧。」

<br/>
<br/>
雷伊來到我身旁。

<br/>
<br/>
「話說回來，你知道什麽不錯的魔劍嗎？」

<br/>
<br/>
「唔，是要代替伊尼迪歐的嗎？」

<br/>
<br/>
「畢竟澈底斷成兩截了嘛。有時間或許能修好，但看來得暫時用其他劍了。」

<br/>
<br/>
畢竟也不能每次都把席菈找來，請她化身爲劍來使用嘛。

<br/>
<br/>
我想想看，藏寶庫裏有適合雷伊的魔劍嗎？

<br/>
<br/>
「啊，大家早安！」

<br/>
<br/>
遠方傳來叫喚。米莎揮著手朝這裏跑來。

<br/>
<br/>
「真難得呢，大家像這樣一起上學。」

<br/>
<br/>
「是啊，似乎是在路上偶然遇到的唷。」

<br/>
<br/>
雷伊答道。

<br/>
<br/>
「是這樣啊？呵呵呵，不過，感覺還真不錯呢，大家能像這樣一起悠閑地上學。我早上總是一個人走，覺得有點寂寞呢。」

<br/>
<br/>
「原來你這麽怕寂寞啊？」

<br/>
<br/>
「啊哈哈……要保密唷……？」

<br/>
<br/>
他們兩人像這樣邊走邊聊著。

<br/>
<br/>
我們一面享受平穩的生活，一面走向德魯佐蓋多。
<br/>
<br/>
